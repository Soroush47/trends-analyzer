# Google Trends Python API Wrapper

This project is a Python script that connects to the Google Trends API using the `pytrends` library, providing functionalities to fetch and analyze data related to keyword popularity and trending searches.

## Key Features

*   **Interest Over Time by Region:** Displays the search interest for specified keywords across different geographical regions.
*   **Trending Searches:** Lists the most popular search queries in a specific country.
*   **Keyword Suggestions:** Provides related keyword suggestions based on an input keyword.
*   **Data Visualization:** Presents results graphically (as bar charts) for better trend understanding.

## Requirements

To run this script, you need Python and the following libraries:

*   Python 3.6+
*   `pandas`
*   `pytrends`
*   `matplotlib`
